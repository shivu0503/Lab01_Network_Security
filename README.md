# Lab 1: Network Construction using Cisco Packet Tracer

* Designed a routed network using Cisco Packet Tracer.
* Implemented Class A, Class B, and Class C IP addressing.
* Used Cisco ISR4331 router with three Gigabit Ethernet interfaces.
* Connected three Cisco 2960 switches to the router.
* Configured PCs and laptops with static IP addresses.
* Assigned proper subnet masks and default gateways.
* Enabled the third router interface manually.
* Verified inter-network communication using ping.
